# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-03-12

### Added
- Initial release of @page-speed/skins
- JSON-based skin system for component styling
- TypeScript schema definitions
- Runtime utilities for applying skins
- Video player adapter functions
- CDN loading utilities with caching
- 5 video player skins:
  - base.json - Minimal base skin
  - linear-inspired.json - Linear design system inspired
  - minimal-hover.json - Hover-based controls
  - social-card.json - Social media optimized
  - youtube-classic.json - YouTube-inspired design
- Comprehensive test suite (133 tests)
- Full TypeScript support
- jsDelivr CDN support
- Skin validation script
- Complete documentation and examples
